# Projet_Vertigo
Projet synthèse pour le cours de Réalité virtuelle à l'UQAC Printemps 2026

## Contexte
Ce projet a été fait lors du cours de **Réalité virtuelle et environnement immersifs** à l'UQAC en hiver 2026.

## Spécificités
- Unity 6000.3.5f2
- Nécessité d'un casque VR (Testé avec un Meta Quest 3)
- Utilisation d'une planche en bois et de ventilateurs pour améliorer l'immersion

## Description
Le projet vise à confronter le joueur à l’acrophobie (peur des hauteurs) à travers une expérience immersive en réalité virtuelle.

Le joueur est placé dans un environnement urbain en hauteur et doit accomplir une tâche simple, mais rendue difficile par le vertige.

## Gameplay

### Objectif
Récupérer un burger situé à l’extrémité d’une planche suspendue dans le vide et le déposer dans une assiette pour compléter le niveau.

### Déroulement
- Le joueur commence sur le toit d’un bâtiment.
- Il réalise qu’il a oublié son dîner.
- Un burger est visible au bout d’une planche étroite qui dépasse du bord du toit.

### Mécaniques
- **Téléportation libre sur le toit**  
  Le joueur peut se déplacer librement sur toute la surface du toit via un système de téléportation.

- **Déplacement physique sur la planche**  
  La téléportation est désactivée sur la planche.  
  Le joueur doit **réellement marcher** sur la planche physique pour avancer, renforçant l’immersion et la sensation de vertige.

- **Interaction avec les objets**
  - Le joueur peut **prendre le burger**.
  - Il doit ensuite revenir sur le toit et **placer le burger dans son assiette** pour terminer le niveau.

## Mise en place physique
- Une planche réelle est alignée avec celle du jeu pour simuler l’équilibre.
- Des ventilateurs sont utilisés pour recréer la sensation de vent en hauteur.

## Expérience visée
Créer une forte sensation de vertige et d’inconfort contrôlé, en combinant :
- la hauteur virtuelle,
- le déplacement physique réel,
- et les sens (vent, planche).
